# Data_Generation_Using_Modelling_And_Simulation

# 1. Methodology

Generating random sample for simulation ----> Applying models on the generated data ----> Graphical display of data and evaluated parameters 

# 2. Description

- Simulator: Gekko  
- Dataset: Generated using simulation
- No. of Simulation: 1000
- ML Models: Linear Regression, Ridge, Lasso, Decision Tree, Random Forest, SVR, KNN
- Output: Table

# 3. Objectives

- Using simulation to generate dataset
- Applying ML models to compare different models
- Graphical Display of data and result.
